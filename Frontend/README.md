This is MERN simple Chatbot project that will be talking to you, you just simply type a prompt and it will reply you.

About Project:--
1). It has simple UI using tailwindcss classes, react icons.
2). It uses axios for sending user text to backend.
3). In backend it is using Node JS and Express JS and mongoose database and all those tech that is used to create backend server.
